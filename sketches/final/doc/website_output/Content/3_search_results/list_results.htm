Displaying search results as a list

The default view of search results is a scatter-plot graph. To change the view, click List under the Display As options that appear at the top of the screen.

The following image shows the results for the subject search "United States Foreign Relations 20th Century" displayed as a list. This image show only the top part of the screen.

Use the mouse or arrow keys to move through the list. As you move through the list, the Inspector displays a detailed record for the currently selected item.

To return to viewing the results as a graph, click Scatter under the Display As options.
