LC class titles on the subject bar

When Haystack first displays the results of a search, titles of the main LC classes appears next to each colored area in the subject bar where there a significant number of results. If the number of results for a given class is small, the label does not appear.

See the following example of the results for the subject search "United States Foreign Relations 20th Century to see class titles on the subject bar.
